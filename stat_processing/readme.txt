Для запуска:
в консоли выполнить -
cd stat_processing
py manage.py runserver

перейти по сформировавшейся ссылке, например:
Starting development server at http://127.0.0.1:8000/



-- в баше в папке с проектом "git config --global core.quotePath false"



Критерий Стьюдента:
2 параметра (только цифры)
Если табличное больше посчитанного то Н0 если наоборот то H1




Менюшка:
  Описательная статистика

  Нормальное распределение
    - Критерий Шапиро-Уилка

  Параметрические критерии
    - t-Критерий Стьюдента

  Не параметрические критерии
    - Критерий знаков
    - Корреляция по Спирману
    - Критерий Уилкоксона
