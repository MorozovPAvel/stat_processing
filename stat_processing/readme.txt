Для запуска:
в консоли выполнить -
cd stat_processing
py manage.py runserver

перейти по сформировавшейся ссылке, например:
Starting development server at http://127.0.0.1:8000/



-- в баше в папке с проектом "git config --global core.quotePath false"



Критерий Стьюдента:
2 параметра (только цифры)
Для каждого столбика вычисляем:
- среднее значение (float 2 знака)
- стандартную ошибку
    находим среднее значение
    находим откланение от среднего (каждое значение - среднее)
    возводим откланения в квадрат
    находим сумму квадратов
    сумму квадратов делим на количество значений в столбце
    извлекаем корень из получившегося числа.

- какая-то хз че за формула, решим потом
