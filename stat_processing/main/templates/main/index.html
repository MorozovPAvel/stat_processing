«МедМатСтат» – это облачное цифровое средство, предназначенное для статистической обработки данных разработанное специально для студентов медицинского вуза и медицинских работников. В медицине важна точность и доступность информации, поэтому в программу встроены необходимые критерии, которые позволяют решить широкий круг медицинских задач. Сложные расчеты скрыты от глаз пользователей, что позволяет сосредоточиться на главном: интерпретации результатов и принятии решений, на основе полученных выводов.

На главном экране представлены два алгоритма выбора статистического критерия, один из которых учитывает типы выборок, другой - специфику решаемой задачи.

При решении любой статистической задачи или проведении статистического анализа данных необходимо провести предварительную статистическую обработку, которая также нужна для выбора подходящего критерия. Для этого вы можете воспользоваться разделами «Описательная статистика» и «Нормальное распределение» в меню.

Мы стремимся сделать статистику доступной для всех, кто работает в области медицины, помогая принимать обоснованные решения на основе полученных данных.
